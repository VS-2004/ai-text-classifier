# AI vs Human Text Classifier

Binary text classification pipeline detecting AI-generated vs human-written
content across social, news, and academic domains. 8-week capstone project.

## Project Structure
- `data/` — raw dataset
- `scripts/` — phase-by-phase Python scripts
- `outputs/figures/` — generated plots and result summaries
